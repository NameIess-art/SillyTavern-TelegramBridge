Telegram bridge plugin for SillyTavern.

Configuration file:
- `data/_plugins/telegram-bridge/config.json`

Status endpoints after login:
- `GET /api/plugins/telegram-bridge/status`
- `GET /api/plugins/telegram-bridge/config`
- `GET /api/plugins/telegram-bridge/chats`
- `POST /api/plugins/telegram-bridge/config`
- `POST /api/plugins/telegram-bridge/select-chat`
- `POST /api/plugins/telegram-bridge/reset`
